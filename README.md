# 🚖 OLA Data Analytics Project

## 📌 Project Overview

This project focuses on analyzing OLA ride data to extract meaningful insights and present them through interactive dashboards. It demonstrates the complete data analytics workflow — from raw data extraction to visualization.

The project uses **SQL, Excel, and Power BI** to solve real-world business problems and generate actionable insights.

---

## 🎯 Objectives

* Analyze ride booking trends and patterns
* Understand customer and driver behavior
* Identify cancellation reasons
* Evaluate revenue and payment methods
* Build an interactive dashboard for decision-making

---

## 🛠️ Tools & Technologies Used

* **SQL** – Data extraction, cleaning, and transformation
* **Microsoft Excel** – Data analysis and preprocessing
* **Power BI** – Dashboard creation and data visualization

---

## 🔍 Project Workflow

### 1️⃣ SQL – Data Extraction & Preparation

* Queried large datasets efficiently
* Cleaned and transformed raw data
* Handled missing and inconsistent values
* Structured data for analysis

---

### 2️⃣ Power BI – Dashboard Creation

* Built interactive dashboards
* Created KPIs like total bookings, revenue, and cancellations
* Visualized trends using charts and graphs
* Designed user-friendly navigation

## 📊 Dashboard Features

### 🔹 Overall Analysis

* Total Bookings: **15,649**
* Total Booking Value: **5M**
* Ride Volume Trends over time
* Booking Status Breakdown

### 🔹 Vehicle Type Insights

* Booking and revenue by vehicle type
* Average distance travelled
* Performance comparison across categories

### 🔹 Revenue Analysis

* Revenue by payment methods (Cash, UPI, Card)
* Daily ride distance trends

### 🔹 Cancellation Analysis

* Customer cancellation reasons
* Driver cancellation reasons
* Cancellation distribution insights

### 🔹 Ratings Analysis

* Customer ratings by vehicle type
* Driver ratings comparison

---

## 📈 Key Insights

* Most revenue is generated through **Cash and UPI payments**
* **Mini and Prime Plus** categories show high usage
* Cancellation reasons highlight operational inefficiencies
* Ratings remain consistent across most vehicle types
